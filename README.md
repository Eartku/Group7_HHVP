# Web_Project_Group7 – Bonsai Shop Website

> *Where bonsai meets code.*  
> Một website bán và trưng bày bonsai, được xây dựng với tinh thần chậm – tinh tế – có chiều sâu, giống như chính nghệ thuật bonsai.

---

## Giới thiệu
**Web_Project_Group7** là dự án xây dựng website cho một **cửa hàng bonsai**

## Tính năng chính
- Trang chủ giới thiệu cửa hàng & tinh thần bonsai  
- Danh sách bonsai (theo loại, dáng, kích thước)
- Trang chi tiết bonsai:  
  - Tên cây  
  - Hình dạng 
  - Giá cả 
  - Mô tả 
- Giỏ hàng & đặt mua (demo)


## Công nghệ sử dụng
- **HTML**
- **CSS**
- **JavaScript**
- Bootstrap 
- PHP cho backend
- Database MySQL
